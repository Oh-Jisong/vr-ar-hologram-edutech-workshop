# vr-ar-hologram-edutech-workshop

> 초등학생 대상 VR·AR·홀로그램 체험형 과학 교육(EduTech) 워크숍 기획 및 운영 프로젝트

---

## 프로젝트 소개

본 프로젝트는 **VR(가상현실), AR(증강현실), 홀로그램 기술**을 주제로  
초등학생을 대상으로 진행한 **체험 중심 과학 교육 워크숍**입니다.

단순 이론 설명이 아닌,

- 흥미 유발 퀴즈  
- 실생활 사례 연결  
- 직접 제작 및 실험 활동  
- 참여형 정리 활동  

을 결합하여 **아이들이 직접 체험하며 기술 원리를 이해하도록 설계**하였습니다.

본 프로젝트는 교육 봉사 활동의 일환으로 직접 기획·제작·운영되었습니다.

---

## 프로젝트 목표

- 미래 기술(VR·AR·홀로그램)을 초등 눈높이에 맞게 전달  
- 체험 기반 학습을 통한 이해도 향상  
- 참여형 수업 구조 설계  
- STEM 교육 및 EduTech 콘텐츠 제작 경험 축적

## 수업 구조 (Learning Flow)
```
도입 퀴즈
→ 경험 나누기
→ VR 원리 설명
→ VR 카드보드 제작 및 체험
→ AR 원리 설명
→ AR 큐브 체험
→ 홀로그램 원리 설명
→ 홀로그램 피라미드 제작
→ 마무리 퀴즈 및 정리
```
