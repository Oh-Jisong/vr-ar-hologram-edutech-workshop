# VR·AR·홀로그램 체험형 과학 수업 계획서

---

## 수업 개요

- 수업명: VR·AR·홀로그램으로 만나는 미래 테크놀로지 체험
- 대상: 초등학생
- 수업 형태: 체험형 STEM 교육 워크숍
- 소요 시간: 약 60분
- 운영 방식: 이론 + 제작 + 체험 + 퀴즈 결합형 수업

---

## 학습 목표

본 수업을 통해 학생들은 다음을 달성하도록 설계되었습니다.

- VR(가상현실)과 AR(증강현실)의 차이를 설명할 수 있다.
- 입체 영상이 만들어지는 기본 원리를 이해한다.
- 홀로그램 영상이 공중에 떠 보이는 이유를 설명할 수 있다.
- 미래 기술에 대한 흥미와 과학적 사고를 기른다.

---

## 준비물

### 이론 수업 자료

- 수업용 PPT 자료
- VR / AR / 홀로그램 원리 설명 이미지

### 체험 활동 키트

- VR 카드보드 키트
- AR 큐브 박스
- 홀로그램 피라미드 필름
- 스마트폰 또는 태블릿

### 기타 준비물

- 필기구
- 퀴즈 활동지 (선택)
- 꾸미기 스티커 (참여도 향상용)

---

## 수업 전체 흐름
```
도입 퀴즈
→ 경험 나누기
→ VR 원리 설명
→ VR 카드보드 제작 및 체험
→ AR 원리 설명
→ AR 큐브 체험
→ 홀로그램 원리 설명
→ 홀로그램 피라미드 제작
→ 마무리 퀴즈 및 정리
```


---

## 단계별 수업 진행

### ① 도입 — 흥미 유발 (약 5분)

#### 활동 내용

- 미래 기술 퀴즈 진행
- 학생들의 기존 경험 유도 질문

#### 예시 질문

- 3D 영화 본 적 있나요?
- 포켓몬GO 해본 적 있는 친구?
- 화면에서 물체가 튀어나온 것처럼 느껴본 적 있나요?

#### 목적

- 수업 집중도 확보
- 학생 참여 분위기 형성
- 기존 경험 기반 연결

### ② VR 원리 설명 (약 8분)

#### 핵심 개념

- 양쪽 눈에 다른 영상을 보여주면 뇌가 입체로 인식함
- 렌즈 + 스마트폰 화면 구조 이해

#### 설명 포인트

- VR은 진짜 세상이 아니라 눈을 속이는 기술
- 깊이감은 뇌의 착시 효과

### ③ VR 카드보드 제작 및 체험 (약 10분)

#### 활동 내용

- 카드보드 조립
- VR 영상 시청 체험

#### 학습 포인트

- 렌즈 역할 이해
- 입체감 체험 직접 경험


### ④ AR 원리 설명 (약 7분)

#### 핵심 개념

- 카메라 화면 위에 가상 객체를 합성
- 마커 인식을 통한 위치 추적

#### 설명 포인트

- AR은 현실 세계 위에 정보를 덧붙이는 기술
- 카메라가 패턴을 인식하는 원리 소개

### ⑤ AR 큐브 체험 (약 8분)

#### 활동 내용

- AR 큐브 조립
- 앱 실행 후 캐릭터 관찰

#### 관찰 활동

- 큐브 회전
- 거리 변화
- 각도 변화

### ⑥ 홀로그램 원리 설명 (약 6분)

#### 핵심 개념

- 빛 반사를 이용한 시각적 착시
- 투명 필름 구조 설명

#### 설명 포인트

- 실제로 공중에 존재하는 것이 아님
- 빛 반사로 떠 보이게 만드는 원리


### ⑦ 홀로그램 피라미드 제작 (약 8분)

#### 활동 내용

- 피라미드 필름 접기
- 스마트폰 영상 실행
- 공중 영상 관찰

#### 학습 포인트

- 반사 위치 확인
- 중심 영상 생성 관찰

### ⑧ 마무리 퀴즈 및 정리 (약 8분)

#### 복습 퀴즈

- VR / AR / 홀로그램 개념 확인
- OX 퀴즈 및 선택형 문제

#### 정리 질문

- 오늘 배운 기술의 핵심 원리는 무엇인가?
- 일상생활에서 사용되는 예시 찾기

- 교구 파손 방지 안내
