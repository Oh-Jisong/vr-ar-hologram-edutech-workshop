# VR·AR·홀로그램 체험 수업 PPT 구성안

---

## PPT 개요

- 대상: 초등학생
- 목적: 미래 기술 개념 이해 + 체험 중심 학습
- 구성 방식: 짧은 문장 + 시각 자료 + 참여형 슬라이드
- 권장 분량: 약 18~22 슬라이드

---

## 전체 슬라이드 흐름
```
표지
→ 오늘 배울 내용
→ 도입 퀴즈
→ 경험 나누기
→ VR 원리
→ VR 제작 및 체험
→ AR 원리
→ AR 체험
→ 홀로그램 원리
→ 홀로그램 제작
→ 마무리 퀴즈 및 정리
```


---

## 슬라이드 구성 상세


### 1️⃣ 표지

#### 내용

- 제목: VR·AR·홀로그램으로 만나는 미래 테크놀로지 체험
- 부제: 직접 만들고 체험하는 미래 기술 수업
- 시각 자료: VR 고글, AR 캐릭터, 홀로그램 이미지


### 2️⃣ 오늘 배울 내용

#### 구성

- VR: 완전히 가상의 세계
- AR: 현실 위에 가상 정보 추가
- 홀로그램: 공중에 떠 보이는 영상

#### 목적

- 수업 전체 흐름 미리 제시
- 학습 목표 인식


### 3️⃣ 도입 퀴즈

#### 문제 예시

- 휴대폰 화면에서 포켓몬이 방 위에 나타나는 기술은?
  - VR / AR / 홀로그램

#### 목적

- 흥미 유발
- 학생 참여 유도


### 4️⃣ 경험 나누기

#### 질문 예시

- 3D 영화 본 적 있나요?
- 얼굴 필터 사용해 본 적 있나요?
- VR 고글 써본 적 있나요?

#### 목적

- 기존 경험 연결
- 분위기 활성화


### 5️⃣ VR 원리

#### 핵심 내용

- 양쪽 눈에 다른 화면 제공
- 뇌의 입체 인식 착시

#### 시각 자료

- 양안 시차 그림
- VR 카드보드 구조 그림


### 6️⃣ VR 카드보드 제작 및 체험

#### 구성

- 카드보드 조립 순서
- 스마트폰 삽입 방법
- 체험 영상 실행

#### 안전 안내

- 어지러움 발생 시 즉시 중단
- 사용 시간 제한


### 7️⃣ AR 원리

#### 핵심 내용

- 카메라 화면 + 가상 객체 합성
- 마커 인식 원리

#### 시각 자료

- 큐브 마커 인식 예시
- AR 화면 합성 그림


### 8️⃣ AR 큐브 체험

#### 구성

- 큐브 조립
- 앱 실행
- 캐릭터 위치 변화 관찰

#### 관찰 포인트

- 회전
- 거리 변화
- 각도 변화


### 9️⃣ 홀로그램 원리

#### 핵심 내용

- 빛 반사 기반 시각적 착시
- 피라미드 구조 설명

#### 시각 자료

- 홀로그램 피라미드 구조 그림
- 반사 경로 이미지


### 🔟 홀로그램 제작 및 체험

#### 구성

- 피라미드 필름 접기
- 스마트폰 위 설치
- 영상 실행

#### 관찰 포인트

- 중심 영상 생성
- 반사 위치 확인


### 1️⃣1️⃣ 마무리 퀴즈

#### 문제 유형

- OX 퀴즈
- 4지선다 문제
- 빈칸 채우기

#### 목적

- 개념 복습
- 학습 성과 확인

### 1️⃣2️⃣ 정리 및 마무리

#### 내용

- 오늘 배운 기술 요약
- 생활 속 활용 예 다시 확인
- 학생 소감 공유
